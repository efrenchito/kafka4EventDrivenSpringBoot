🚀 How to Run Apache Kafka on Windows
=======================================================================================================================================

⚓ https://www.confluent.io/blog/set-up-and-run-kafka-on-windows-linux-wsl-2/

Windows isn’t the recommended platform for running Kafka with production workloads.
 -> However for trying out Kafka, it works just fine.
Thanks to the Windows Subsystem for Linux 2 (WSL 2), now we can run Apache Kafka® on Windows.


Set up your environment 
  +  Install Windows Subsystem for Linux
    - Enable WSL 2
    - Install Java 
    - Download Kafka

Make sure you’re running Windows 10 version 21H1 or later, or Windows 11 21H2 or later. 
To check your version, navigate to Settings > System > About. 
In the Windows specifications section, find Version.
(Ensure that you have all updates to your version installed.)


📝 Enable WSL 2
-----
To install the WSL 2 feature, open PowerShell as an administrator, and run the following command:

$ wsl --install
~
This command installs the Microsoft Store version of WSL and automatically selects the WSL 2 version. 
Also, it installs the default Linux distro, which is Ubuntu 22.04 as of this writing.
Tip: If you already have the non-Store version of WSL installed, you can run the wsl –update command to get it.

 -> Reboot your machine.


📝 Install Java


📝 Download Kafka
-----
⚓ https://kafka.apache.org/downloads

wget https://dlcdn.apache.org/kafka/3.5.0/kafka_2.13-3.5.0.tgz
..
tar -xzf kafka_2.13-3.5.0.tgz
cd kafka_2.13-3.5.0
